# Breast Cancer Survival Classification

A machine learning classification project that predicts patient survival outcomes using clinical and demographic breast cancer data.

This project is designed as both a portfolio project and a practical machine learning learning journey. The goal is to demonstrate the complete machine learning workflow, from exploratory data analysis and preprocessing to model training, evaluation, and prediction.

## Project Objectives

* Load and explore the dataset
* Perform exploratory data analysis (EDA)
* Visualize important patterns and relationships
* Clean and preprocess the data
* Encode categorical features
* Scale numerical features where appropriate
* Split the dataset into training and testing sets
* Train multiple classification models
* Tune model parameters
* Evaluate model performance using classification metrics
* Compare different models
* Generate predictions on unseen data
* Draw conclusions from the results

## Dataset

The dataset contains demographic and clinical information about breast cancer patients, including factors such as age, tumor characteristics, cancer stage, hormone receptor status, and survival information.

## Target Variable

The target variable is **Status**, which indicates whether a patient was:

* Alive
* Dead

The objective is to predict this outcome using the available patient features.
